## 4. How to Use

### Option A: Inline SVGs (paste directly into HTML)

Replace your current feature icons in the HTML files with the SVGs above. Each uses `fill="#ffa800"` to match your gold accent color (or `fill="currentColor"` for text-colored icons).

### Option B: `<img>` tags (fastest)

```html
<img src="https://api.iconify.design/mdi:fan.svg" alt="ريشة نحاس" width="40" height="40">
```

### Option C: Iconify JS (best for dev workflow)

```bash
npm install @iconify/iconify
```

Then in your HTML:

```html
<span class="iconify" data-icon="mdi:fan" data-width="40" data-height="40"></span>
```

In your JS entry point:

```js
import 'iconify-icon';
```
